Herleitung der Wellengleichungen elektromagnetischer Wellen aus den Maxwellgleichungen

Maxwellgleichungen

Gaussches Gesetz für Elektrische Felder (1): \begin{aligned} \nabla \cdot \vec {E} = \frac{ \rho }{ \epsilon_0 } \end{aligned} wird hier zu \begin{aligned} \nabla \cdot \vec {E} = 0 \quad , \end{aligned} da die Ladungsdichte $ \rho = 0 $, da keine Ladung vorhanden ist.

Gaussches Gesetz für Magnetfelder (2): \begin{aligned} \nabla \cdot \vec {B} = 0 \end{aligned}

Induktionsgesetz (3): \begin{aligned} \nabla \times \vec {E} = - \frac{ \partial \vec {B} }{ \partial t } \end{aligned}

Ampere-Maxwell'sches Druchflutungsgesetz (4) \begin{aligned} \nabla \times \vec {B} = \mu_0 \mu_r (\vec {j} + \epsilon_0 \epsilon_r \frac{ \partial \vec {E} }{ \partial t }) \end{aligned} wird hier, im Fall des Vakuums ($ \epsilon_r = \mu_r = 1 $) und der Abwesenheit von äußeren Strömen ($ \vec {j} = 0 $) zu: \begin{aligned} \nabla \times \vec {B} = \mu_0 \epsilon_0 \frac{ \partial \vec {E} }{ \partial t } \end{aligned}

Differentialgleichung des Elektrischen Feldes

Wir beginnen mit dem Induktionsgesetz (3): \begin{aligned} \nabla \times \vec {E} = - \frac{ \partial \vec {B} }{ \partial t } \end{aligned} Dann bilden wir die Rotation ($ \nabla \times (...) $ ) auf beiden Seiten: \begin{aligned} \nabla \times (\nabla \times \vec {E}) = \nabla \times (- \frac{ \partial \vec {B} }{ \partial t }) \end{aligned} Die Rotation kann in die parielle Zeitableitung hineingezogen werden: \begin{aligned} \nabla \times (\nabla \times \vec {E}) = - \frac{ \partial }{ \partial t } (\underbrace{ \nabla \times \vec {B}}_{=\mu_0 \epsilon_0 \frac{ \partial \vec {E} }{ \partial t } } ) \end{aligned} Wir erkennen auf der rechten Seite das Durchflutungsgesetz (4). Außerdem verwenden wir auf der linken Seite die Vektoridentität $ \nabla \times (\nabla \times \vec {E}) = \nabla \cdot (\nabla \cdot \vec {E}) - \nabla^2 \cdot \vec {E}$ : \begin{aligned} \nabla \cdot (\underbrace{\nabla \cdot \vec {E}}_{=0} ) - \nabla^2 \cdot \vec {E} = - \mu_0 \epsilon_0 \frac{ \partial^2 \vec {E} }{ \partial t^2 } \end{aligned} Wir erkennen die 1. Maxwellsche Gleichung auf der linken Seite und der Ausdruck reduziert sich zu einer typischen Wellengleichung: \begin{aligned} \nabla^2 \vec {E} - \mu_0 \epsilon_0 \frac{ \partial^2 \vec {E} }{ \partial t^2 } = 0 \end{aligned} Wellengleichungen (wie z.B. in der Mechanik) sehen immer folgendermaßen aus: \begin{aligned} \nabla^2 \vec {A} - \frac{ 1 }{ v^2 } \frac{ \partial^2 \vec {A} }{ \partial t^2 } = 0 \end{aligned} Daher gilt hier $ \mu_0 \epsilon_0 = 1/v^2 $ $ \implies v = 1/\sqrt{\mu_0 \epsilon_0} = c $. Elektromagnetische Wellen breiten sich im Vakuum mit Lichtgeschwindigkeit aus.

Eine übliche Form der Wellen-Differentialgleichung für $ \vec {E} $ ist: \begin{aligned} \nabla^2 \vec {E} - \frac{ 1 }{ c^2 } \frac{ \partial^2 \vec {E} }{ \partial t^2 } = 0 \end{aligned}
